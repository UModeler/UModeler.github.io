Clip Tool ¶

Cuts away the parts of the selected polygons under the Clip polygon, which has to be selected after pressing Clip button.

Steps¶

  1. Select polygons.
  2. Go to Clip Tool
  3. Move the cursor to the polygon which will clip the selected polygons and press LMB
  4. You can check out that the selected polygons are clipped.
The part of the selected polygons below the clipped polygon are removed.
