UModeler¶

Korean Site

Introduction¶

​UModeler is a modeling tool in Unity which makes it easier to create various geometric models from prototype levels to complex static meshes, without the external DCC tools.​ Those geometric figures made with UModeler can also be used for collision, navigation mesh, trigger zone etc.​ This will help you a lot in creating great game worlds.

Key Features¶

SketchUp Style Modeling
Any polygons can be drawn easily and you can turn them into 3D shapes in intuitive ways.
Blender Style Modeling
If you want to create complicated shapes like buildings, vehicles, props, etc, you can do it with the advanced tools like extrude, inset, boolean, bevel, slice, loop slice, mirror, follow tool, etc.

Basic and Pro¶

UModeler Basic
The basic version includes the core features enough to create simple meshes and levels.
UModeler Pro
The Pro version has lots of advanced tools such as bevel, boolean, mirror, slice, loop slice, follow, and much more. With Pro version, you can make complicated shapes for real assets without leaving Unity.

Note

If you want to know the difference more between Basic and Pro, see Basic and Pro Comparison.
