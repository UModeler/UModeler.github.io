Introduction
===============

UModeler is a ultimate and intuitive modeling extension that is available on Unity Editor. 

UModeler has brought major features of SketchUp so that you can draw 2D shapes on any polygons in easy ways and turn them into 3D shapes intuitively.

Moreover UModeler has also adopted useful features of popular DCC tools like Max and Blender etc so that empowers you to create complicated and rich mesh assets as well as game levels with ease without any help of the external DCC tools.

User Interface
================

+---------------------------------------+----------------------------------------------------------+
| ``SPACE``                             | Confirms the current action.                             |
+---------------------------------------+----------------------------------------------------------+
| ``ESC``                               | Cancels the current function or Exit the current tool.   |
+---------------------------------------+----------------------------------------------------------+
| ``ENTER``                             | Enables Search Edit Box.                                 |
+---------------------------------------+----------------------------------------------------------+
| ``CTRL + Z``                          | Undo                                                     |
+---------------------------------------+----------------------------------------------------------+
| ``CTRL + Y``                          | Redo                                                     |
+---------------------------------------+----------------------------------------------------------+
| ``W``                                 | Translation Gizmo                                        |
+---------------------------------------+----------------------------------------------------------+
| ``E``                                 | Rotation Gizmo                                           |
+---------------------------------------+----------------------------------------------------------+
| ``R``                                 | Scale Gizmo                                              |
+---------------------------------------+----------------------------------------------------------+
| ``LMB``                               | Selects or moves several elements.                       |
+---------------------------------------+----------------------------------------------------------+
| ``LMB Drag``                          | Selects or moves several elements.                       |
+---------------------------------------+----------------------------------------------------------+
|``SHIFT + LMB`` or ``LMB Drag``        | Special action depending on the current tool             |
+---------------------------------------+----------------------------------------------------------+
|``CTRL + LMB`` or ``LMB Drag``         | Special action depending on the current tool             |
+---------------------------------------+----------------------------------------------------------+
``LMB`` - Left Mouse Button
``RMB`` - Right Mouse Button
``MMB`` - Middle Mouse Button
