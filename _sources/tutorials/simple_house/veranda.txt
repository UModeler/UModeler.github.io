*************
Veranda
*************

 1.With Parallel tool under Drawing group, draw a parallel line from the bottom edge of the side wall.
 It should be 1.65m apart from the bottom edge. You can enter 1.65m in Distance property in the inspector.  
  .. figure:: /images/tutorials/simple_house/veranda_0.jpg

 2.One more edge which is 1.5m apart from the bottom edge should be added likewise with Parallel tool.

  .. figure:: /images/tutorials/simple_house/veranda_1.jpg

 3.You should pull a rectangle between 1.5m and 1.65m on the side wall by 1.5m to make a plank as below.

  .. figure:: /images/tutorials/simple_house/veranda_2.jpg

 4.Draw a rectangle using Rectangle tool at the corner of the back of the plank made in step 3.

  .. figure:: /images/tutorials/simple_house/veranda_3.jpg

 5.Clone and place the rectangle made in step 4 using Clone tool. The cloned rectangle should be located at 3.9m apart in the opposite direction.

  .. figure:: /images/tutorials/simple_house/veranda_4.jpg

 6.Draw another rectangle at the other corner of the back of the plank.

  .. figure:: /images/tutorials/simple_house/veranda_5.jpg 

 7.Pull the rectangle drawn in step 6 with Extrude leaving "Heiht Alignment" enabled. 
 In order to pull the rectangle up to the bottom of the house, drag the mouse cursor to the bottom.
  
  .. figure:: /images/tutorials/simple_house/veranda_6.jpg

 8.Click LMB holding SHIFT in Extrude Tool on the two rectangles drawn in step 4 and 5 to make them pillars.

  .. figure:: /images/tutorials/simple_house/veranda_7.jpg

 9.Now you'll make a handrail on the plank. Using Inset tool, make a smaller rectangle by 1m on the plank as follows.

  .. figure:: /images/tutorials/simple_house/veranda_8.jpg  

 10.Go to Erase tool. Remove an inner edge.

  .. figure:: /images/tutorials/simple_house/veranda_9.jpg  

 11.Connect an cut point behind with a point on the wall using Polyline tool.

  .. figure:: /images/tutorials/simple_house/veranda_10.jpg  

 12.Do likewise at the opposit side.

  .. figure:: /images/tutorials/simple_house/veranda_11.jpg

 13.Select Extrude Tool. Pull up the part of the handrail on the plank by 0.85m. At this point "Leave Side Edges" needs to be on.

  .. figure:: /images/tutorials/simple_house/veranda_12.jpg

 14.Pull up the part you pulled up in step 13 by 0.1m again.
 
  .. figure:: /images/tutorials/simple_house/veranda_13.jpg
  
 15.Push each middle rectangle of the handrail till it disappears using Extrude tool leaving "Boundary Check" and "Leave Side Edges" enabled.
  .. figure:: /images/tutorials/simple_house/veranda_13.5.jpg

 16.Remove unecessary edges created in step 13 and 14.
 
  .. figure:: /images/tutorials/simple_house/veranda_14.jpg
  .. figure:: /images/tutorials/simple_house/veranda_15.jpg
  .. figure:: /images/tutorials/simple_house/veranda_16.jpg
  .. figure:: /images/tutorials/simple_house/veranda_17.jpg

 17.Draw a rectange whose size is 1m x 1m to be fitted into the corner boundary on the plank.
 
  .. figure:: /images/tutorials/simple_house/veranda_18.jpg

 18.With Inset tool, Create a 0.02m smaller rectangle than the rectangle drawn in step 16.
 
  .. figure:: /images/tutorials/simple_house/veranda_19.jpg

 19.Remove edges surrounding the rectangle drawn in step 17.
 
  .. figure:: /images/tutorials/simple_house/veranda_20.jpg

 20.Move the camera to the opposite side. Draw a parallel edge with the right edge on the plank. The parallel edge should be 0.02m apart from the right edge.
 
  .. figure:: /images/tutorials/simple_house/veranda_21.jpg

 21.Move the camera to the opposite side again. Go to Clone tool and move the mouse cursor so that the rectangle drawn in step 17 will be highlighted. 
 And drag the mouse to move the highlighted rectangle up to the edge drawn in step 19. Make the line straight and release LMB and set Number property in the inspector to 25.
 
  .. figure:: /images/tutorials/simple_house/veranda_22.jpg
  
 22.Using Parallel tool, draw an edge as follows. It should be 0.02 apart from the wall.
 
  .. figure:: /images/tutorials/simple_house/veranda_23.jpg
  
 23.Go to Clone tool to make 4 more rectangles from the rectangle at the corner as below.
 
  .. figure:: /images/tutorials/simple_house/veranda_24.jpg
  
 24.Do in the opposite side as you did in step 23. 
 
  .. figure:: /images/tutorials/simple_house/veranda_25.jpg
  
 25.Remove all edges execpt the rectangles created with Clone Tool as below.
  
  .. figure:: /images/tutorials/simple_house/veranda_26.jpg
  .. figure:: /images/tutorials/simple_house/veranda_27.jpg
  .. figure:: /images/tutorials/simple_house/veranda_28.jpg
  
 26.With Extrude tool, pull up one of the rectangles to the upper handrail. "Boundary Check" property needs to be on while doing this.
 
  .. figure:: /images/tutorials/simple_house/veranda_29.jpg
  
 27.Click LMB holding Shift on all the rest rectangles to repeat step 26.
 
  .. figure:: /images/tutorials/simple_house/veranda_30.jpg
  
 28.Veranda has been complete.
 
  .. figure:: /images/tutorials/simple_house/veranda_31.jpg
