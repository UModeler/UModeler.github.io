Introduction
============

​UModeler is a modeling tool in Unity to make it easy to create various geometry models such as prototype levels and complex static meshes without the external DCC tools.​
Those geometries made with UModeler can also be used for collision, navigation mesh and trigger zone etc.​
This will help you a lot to create great game worlds

Key Features
============

 SketchUp style Modeling
	Any polygons can be drawn easily and you can make them 3D shapes in intuitive ways.

 Blender style Modeling
	If you want to create complicated shapes like buildings, vehicles and props etc, you can do it with the advanced tools like extrude, inset, boolean, bevel, slice, loop slice, mirror and follow tool etc.
	
Basic and Pro
=============

 UModeler Basic
	Basic version includes the core features enough to create simple meshes and levels.
	This is free.
	
 UModeler Pro
	Pro version has lots of advanced tools such as bevel, boolean, mirror, slice, loop slice, follow and much more.
	With Pro version you can make complicated shapes for real assets without leaving Unity.
	This is a paid extension.
	
 .. note::

	  If you want to know the difference more between Basic and Pro, see :doc:`basic_vs_pro_comparison`.
