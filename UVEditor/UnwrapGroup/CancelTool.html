Cancel Tool ¶

Cancel will remove the selected polygons from UV Islands and the UVs of them will be reset to have UVs created automatically.

Steps¶

  1. Select polygons which are unwrapped.
  2. Go to Cancel Tool in Unwrap Group

Selects unwrapped polygons.

Cancel Tool gives the polygons automatic generated UVs so the polygons are removed from UV Editor.
