Export Tool ¶

Exports UV outlines to a .png formatted file to be used for an artist which is painting your UV texture.

Properties¶

Line Color

The color of UV outlines.

Transparent Background

If enables, then the background will be transparent and Background Color will be ignored.

Background Color

This will be enabled only when Transparent Background is on.

Resolution

The size of the .png image file.
