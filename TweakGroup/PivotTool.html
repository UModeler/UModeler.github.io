Pivot Tool Â¶

Sets a position of a pivot of the current object. The current position of 3D cursor will be a pivot pos immediately after selecting this tool.

Tool Handle Position should be set as Pivot to check out the pivot position of a game object.

Position the 3D cursor and select Pivot Tool. Then the 3D cursor pos will be the pivot.
